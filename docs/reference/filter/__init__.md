# `graflo.filter`

Filter expressions for AQL, Cypher, SQL, nGQL, GSQL, and Python evaluation.

**Bindings / SQL:** use `TableConnector.filters` or `SelectSpec.where` with logical
operator shorthand (`OR:`, `AND:`, …). See
[Bindings filter cookbook](../../concepts/table_connector_views.md#bindings-filter-cookbook-tableconnectorfilters).

::: graflo.filter
